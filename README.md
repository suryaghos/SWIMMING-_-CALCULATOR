# 🏊 GAF Swim Coach — Calculator, Tracker & Workout Planner

A mobile-first, single-file Progressive Web App (PWA) designed for young swimmers aged **8–14**. GAF Swim Coach helps coaches and parents track, analyze, and improve swim performance — all from the browser, with no installation required.

---

## ✨ Features

- 🏊 **Swim Calculator** — Input distance, time, and stroke count to calculate pace, SWOLF score, stroke rate, and performance rating
- 👤 **Multi-Swimmer Profiles** — Add and switch between multiple swimmer profiles, each with their own history and stats
- 📊 **Session History & Compare** — Log sessions and compare performance over time across swimmers
- 🏋️ **Workout Planner** — Generate and track structured swim workouts
- 🏅 **Badges & Star Ratings** — Motivational tools to celebrate swimmer achievements
- 📤 **Share Card** — Generate a shareable summary of a swimmer's session
- 🌙 **Dark Mode** — Full light/dark theme support
- 📱 **PWA Support** — Installable on Android and iOS for an app-like offline experience

---

## 🚀 Getting Started

No installation or setup needed. Just open the file in any modern browser:

```bash
# Clone the repository
git clone https://github.com/yourusername/gaf-swim-coach.git

# Open the app
open gaf-swim-coach.html
```

Or host it on **GitHub Pages** and access it from any device.

---

## 📱 Install as an App (PWA)

**Android (Chrome):**
1. Open the app in Chrome
2. Tap the banner or menu → *"Add to Home Screen"*

**iOS (Safari):**
1. Open the app in Safari
2. Tap the Share button ↑ → *"Add to Home Screen"*
